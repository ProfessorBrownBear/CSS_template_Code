CSS Flexbox Image Cards Lab

Creating a 3×3 grid with consistent image sizing

Lab Objectives:

This lab demonstrates how to:

  1. Create a 3×3 grid of cards using CSS Flexbox
  2. Maintain consistent image dimensions regardless of source image proportions
  3. Implement a clean, modern card design with hover effects

Key CSS properties used: display: flex, flex-wrap: wrap, object-fit: cover

Key Code Segments Explained

HTML Structure
<div class="cards-container"> <div class="card"> <div class="card-img-container"> <img src="image-path.jpg" alt="Description"> </div> <div class="card-content"> <h3 class="card-title">Card Title</h3> <p class="card-text">Card description text here.</p> <a href="#" class="card-button">Learn More</a> </div> </div> </div>
CSS for Flexbox Grid Layout
/* Container for the entire card grid */ .cards-container { max-width: 1000px; margin: 0 auto; display: flex; /* Makes this a flex container */ flex-wrap: wrap; /* Allows items to wrap to multiple rows */ gap: 20px; /* Creates spacing between cards */ } /* Individual card styling */ .card { /* Each card takes up roughly 1/3 of container width minus gaps */ flex: 0 0 calc(33.333% - 14px); background-color: white; border-radius: 8px; overflow: hidden; box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1); }
CSS for Consistent Image Sizing
/* Image container for consistent sizing */ .card-img-container { width: 100%; /* Full width of card */ height: 200px; /* Fixed height for all images */ overflow: hidden; /* Hide overflow if image is larger */ } /* Image styling to maintain aspect ratio while filling container */ .card-img-container img { width: 100%; /* Full width of container */ height: 100%; /* Full height of container */ object-fit: cover; /* KEY PROPERTY: Makes images cover the area while maintaining aspect ratio */ }

How This Works:

  1. The display: flex and flex-wrap: wrap properties create a flexible grid that allows items to wrap to new rows when they run out of space.
  2. Each card is set to take up exactly 1/3 of the container width (minus some space for gaps) with flex: 0 0 calc(33.333% - 14px).
  3. Setting a specific height: 200px for the image container ensures all images have the same height.
  4. The object-fit: cover property is the key to making images of different dimensions display consistently - it ensures the image covers the entire container while maintaining its aspect ratio (similar to background-size: cover).

Experiment by changing the source images to see how they always maintain the same size in the grid!
